# Umnico Voice Greeting Bot

Автоматически отправляет голосовое приветствие `Салем_1.ogg` всем новым чатам
в боте GoodZhan (telebot, ID 108954) со статусом "Первичный".

## Установка

```bash
pip install requests python-dotenv
```

## Настройка

Создайте файл `.env` рядом с `app.py`:

```
UMNICO_TOKEN=ВАШ_ТОКЕН
```

Токен: **Umnico → Настройки → API Public**

## Запуск

```bash
python app.py
```

## Файлы

| Файл | Описание |
|---|---|
| `app.py` | Основной скрипт |
| `Салем_1.ogg` | Голосовое приветствие |
| `.env` | Токен API (создать самостоятельно) |
