# Umnico Auto Greeting Bot

Автоматически отправляет приветствие (голосовое, видео, фото или документ) всем новым клиентам в указанной интеграции со статусом "Первичный".

## Особенности

- ✅ Отправляет приветствие только **новым клиентам** (по `customer.id`)
- ✅ Поддержка разных типов файлов: audio, video, photo, doc
- ✅ Настраиваемая интеграция и статус через `.env`
- ✅ Обработка всех сетевых ошибок
- ✅ Логирование в файл с UTF-8 кодировкой
- ✅ Graceful shutdown (Ctrl+C)

## Установка

```bash
pip install requests python-dotenv
```

## Настройка

Создайте файл `.env` рядом с `app.py` (используйте `.env.example` как шаблон):

```env
# Umnico API Token (Umnico → Настройки → API Public)
UMNICO_TOKEN=your_api_token_here

# Файл для отправки (путь относительно app.py)
GREETING_FILE=Салем_1.ogg

# Тип файла: audio, video, photo, doc
FILE_TYPE=audio

# ID интеграции (найти в Umnico → Интеграции)
TARGET_SA_ID=108954

# ID статуса (найти в Umnico → Настройки → Статусы)
TARGET_STATUS_ID=958299

# Интервал опроса API в секундах (по умолчанию: 10)
POLL_INTERVAL=10

# Файл для логов (по умолчанию: bot.log)
LOG_FILE=bot.log
```

### Как найти ID интеграции и статуса

1. **ID интеграции (TARGET_SA_ID)**:
   - Откройте Umnico → Интеграции
   - Найдите нужную интеграцию
   - ID можно получить через API: `GET /v1.3/integrations`

2. **ID статуса (TARGET_STATUS_ID)**:
   - Откройте Umnico → Настройки → Статусы
   - ID можно получить через API: `GET /v1.3/statuses`

## Запуск

```bash
python app.py
```

## Типы файлов

Бот поддерживает следующие типы файлов:

- `audio` - аудиофайлы (.ogg, .mp3, .wav)
- `video` - видеофайлы (.mp4, .avi, .mov)
- `photo` - изображения (.jpg, .png, .gif)
- `doc` - документы (.pdf, .docx, .txt)

## Логика работы

1. При запуске бот загружает всех существующих клиентов
2. Каждые N секунд (POLL_INTERVAL) опрашивает API на наличие новых обращений
3. Если находит нового клиента с нужной интеграцией и статусом - отправляет приветствие
4. Каждый клиент получает приветствие **только один раз**, даже если создаст несколько обращений
